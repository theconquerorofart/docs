The Memory skill helps the assistant remember facts, preferences, and context about you. When you save something to memory, the assistant can use it in future conversations.

## When to use it

- Save personal preferences so the assistant tailors responses consistently.
- Store project details you reference often.
- Remind the assistant of past decisions or agreed-upon workflows.
- Recall information you saved in an earlier session.

## Call it from the slash command system

Type `/memory` in the composer, select the skill, then tell the assistant what to remember or ask what it already knows.

## Ask the assistant directly

Pro users can also ask the assistant to use the Memory skill in plain language without opening the slash menu.
