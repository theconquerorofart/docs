You can create custom skills for repeatable actions that are specific to your workflow.

## Create a custom skill

1. Describe what you want the skill to do.
2. The assistant helps you define:
   - **Name** — What the skill is called.
   - **Purpose** — What the skill should help with.
   - **Inputs** — Information the skill needs before it runs.
   - **Behavior** — What the skill should do with those inputs.
3. If the skill requires a secret or external credential, mark it as requiring a secret during creation. A connectivity key is generated for use in **Tool Pairing**.

The assistant checks the custom skill before registering it.

## Use your custom skill

Once forged, the skill appears alongside your other skills. You or the assistant can use it when the task matches what the skill was built to do.

## Requirements

Forging custom skills requires Pro access.
