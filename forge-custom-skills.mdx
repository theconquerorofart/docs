You can create custom skills that the assistant can use, going beyond the built-in slash commands.

## Create a skill

1. Type `/forge` in the composer.
2. Describe what the skill should do. The assistant guides you through defining:
   - **Name** — What the skill is called.
   - **Description** — What it does, so the assistant knows when to use it.
   - **Parameters** — Input values the skill accepts.
   - **Code** — The logic that runs when the skill is called.
3. Provide optional **test arguments** to dry-run the skill.
4. If the skill requires a secret or external credential, you can mark it as **secret required** during creation. A connectivity key is generated for use in **Tool Pairing**.

The assistant validates and dry-runs your skill before registering it.

## Use your skill

Once forged, the skill appears in the slash command menu. You or the assistant can call it like any other skill.

## Requirements

Forging custom skills requires Pro access.
