The Tune skill lets you update an existing forged skill. You can rename it, refine its instructions, update parameters, or modify its code.

## When to use it

- After testing a forged skill, refine its instructions or logic to handle edge cases you discovered during real use.
- Rename a skill to match updated naming conventions or make it easier to find in the command menu.

## Call it from the slash command system

Type `/tune` in the composer, select the skill, then describe what you want to change.

## Ask the assistant directly

Pro users can also ask the assistant to use the Tune skill in plain language without opening the slash menu.
