AscentialxAI offers different access levels. Basic chat with your own provider and core skills are available when you connect a provider. Some features may require a Pro plan or license key.

## Locked features

If you try to access a feature that requires Pro, you'll see a paywall with options to upgrade or redeem a license key.

Feature families that require Pro access:

- **Prompt Library** — Save and reuse prompt templates.
- **Automation Hub** — Build and run multi-step flows.
- **Full access to Browser Skill** — Run advanced browser tasks with Agent Focus.
- **Forge, Tune, and Prune** — Create and manage custom skills.
- **System Prompt customization** — Customize the assistant's behavior.
- **Tool Pairing** — Pair secrets and credentials for forged tools.

## Upgrade

From the paywall, select **Upgrade** to visit the pricing page and choose a plan.

## License redemption

If you have a license key, enter it in the paywall's license field to unlock Pro features.

## Check your plan

Your current plan and access status are shown in **Settings** \> **Account** and on your account dashboard.
