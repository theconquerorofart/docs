AscentialxAI offers basic and pro access levels, so you can start with the core chat experience and unlock advanced workflows when you need them.

## Free tier

Free access includes:

- **Core chat** — Use the side-panel chat experience with your selected model.
- **Core skills** — Use built-in skills such as Memory, Vision, Scraper, and Document.
- **Basic Browser Skill access** — Run basic browser tasks from chat.
- **Context tools** — Attach supported files, images, screenshots, and current-page context.

## Pro tier

Some features are part of the Pro tier and unlock after you upgrade your account.

Pro access includes:

- **Prompt Library** — Save and reuse prompt templates.
- **Automation Hub** — Build and run multi-step flows.
- **Full access to Browser Skill** — Run advanced browser tasks with Agent Focus.
- **Forge, Tune, and Prune** — Create and manage custom skills.
- **System Prompt customization** — Customize the assistant's behavior.
- **Tool Pairing** — Pair secrets and credentials for forged skills.

## Upgrade to Pro

Purchase a Pro license from the pricing page, then redeem the license key through your account access flow.

## Check your plan

Your current plan and access status are shown in **Settings** \> **Account** and on your account dashboard.
