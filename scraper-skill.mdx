The Scraper skill extracts information from the web page you're currently viewing. It can pull visible text, structured data, and page metadata.

## Use Cases

- Capturing page content for research, note-taking, or summarization without manual copy-paste.
- Extracting structured data from a page for further processing in a flow.

## Call it from the slash command system

Type `/scraper` in the composer, select the skill, then specify what data you want to extract.

## Ask the assistant directly

Pro users can also ask the assistant to use the Scraper skill in plain language without opening the slash menu.
