# Overview

> AscentialxAI is a Chrome extension that brings AI-powered chat, skills, and automation to your browser side panel.

## Where you work

The extension lives in Chrome's side panel. From here you can:

- **Chat** — Send prompts, attach context, and get answers from the model you choose.
- **Use skills** — Type `/` to access commands for memory, vision, scraping, documents, browser tasks, automation, and more.
- **Save prompts** — Reuse your best prompts with the Prompt Library.
- **Automate workflows** — Build and run multi-step flows with the Automation Hub.
- **Configure settings** — Manage providers, models, system prompt, tool pairing, and account.

## How it works

You bring your own API keys from supported AI providers. Connect a provider, select a model, and start chatting. The extension works with your current browser tab — you can ask questions about the page, extract data, or perform browser actions.

Slash commands give you access to built-in skills like memory (`/memory`), vision (`/vision`), page scraping (`/scraper`), document analysis (`/document`), and browser tasks (`/browser`). You can also create custom skills with `/forge`.

For repeatable work, the Prompt Library lets you save prompt templates, and the Automation Hub lets you build and run multi-step flows.

## Plans and access

Basic chat and core skills are available when you connect a provider. Some features — like the Prompt Library, Automation Hub, Agent Focus and autonomous browser work, custom skills, and system prompt customization — may require a Pro plan or license.
