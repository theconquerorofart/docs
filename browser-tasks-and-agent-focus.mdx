The assistant can work with the page you're viewing. Use `/browser` or describe a browser task in plain language.

## What you can do

- **Open a URL** — Navigate to a specific page.
- **Inspect the page** — Ask what's visible, find sections, or extract text.
- **Scroll** — Move through the page.
- **Find content** — Locate specific information on the page.
- **Click and type** — Interact with page elements.
- **Extract data** — Pull structured information from the visible page.
- **Write from page context** — Use what you see on the page to compose output.
- **Go back** — Return to the previous page.
- **End session** — Stop browser work and return to chat.

## Agent Focus

When the assistant performs a browser task, an **Agent Focus** overlay appears. This shows that the assistant is working actively with the page. You can press **ESC** at any time to exit Agent Focus and return to normal chat.

## Requirements

Browser tasks work with standard web pages. The assistant may not be able to access Chrome system pages (like `chrome://` URLs) or pages where the extension doesn't have tab access.
