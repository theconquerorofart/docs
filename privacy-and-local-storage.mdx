## Provider keys

API keys you enter are stored through the extension's secure vault path. They are used only when making calls to the provider you selected.

## Chat and extension data

Your chat history, prompts, automation flows, forged skills, memory, and settings are stored in browser and account-scoped extension storage.

## Provider calls

When you send a message or run a flow, the request goes directly to the model provider you selected using your own API key. The extension does not proxy your requests through an intermediary server.

## Factory Reset

Factory Reset removes locally stored extension data, including chats, prompts, automation flows, forged skills, tool pairings, memory, settings, and provider keys stored in the vault.

## Sign out

Signing out ends your current session. The next time you open the extension, you will need to sign in again.
