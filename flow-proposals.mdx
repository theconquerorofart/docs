You can ask the assistant to create a draft automation flow using natural language. Use `/flow` followed by a description of what you want to automate.

## How it works

1. Type `/flow` in the composer and describe the workflow you need (for example, `/flow summarize every page I visit and save it to my notes`).
2. The assistant creates a draft flow and saves it to the **Automation Hub**.
3. Open the Automation tab to review the draft.

## Review and approve drafts

In the Automation Hub, draft flows appear with a **Draft** status. You can:

- **Review** — See the full flow configuration.
- **Edit** — Make changes to steps, triggers, or model settings.
- **Approve** — Save as a live, runnable flow.
- **Reject** — Discard the proposal.
- **Delete** — Remove the draft.

Draft flows do not run automatically. They must be reviewed and approved first.
