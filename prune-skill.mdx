The Prune skill shows you a list of your forged skills along with usage information, and lets you delete skills you no longer need.

## Use Cases

- Cleaning up accumulated forged skills to keep the command menu organized.
- Reviewing which skills are actively used and which have gone dormant.

## Call it from the slash command system

Type `/prune` in the composer, select the skill, then choose which skills to delete.

## Ask the assistant directly

Pro users can also ask the assistant to use the Prune skill in plain language without opening the slash menu.
