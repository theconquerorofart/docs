You can set a custom system prompt to guide the assistant's behavior across all conversations and automation flows.

## Set your system prompt

1. Open **Settings** \> **System Prompt**.
2. Enter your instructions in the textarea.
3. Click **Save**.

The system prompt textarea has a limit of 1,500 characters.

## Reset to default

Click **Reset** to clear your custom prompt and return to the default system prompt.

## How it works

The system prompt is included whenever the assistant responds in chat or runs an automation flow. It sets the overall context and behavior for all interactions.

## Requirements

System prompt customization requires Pro access.
