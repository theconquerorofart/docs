The Document skill processes files you attach to the chat. It can read documents, answer questions about their content, and generate summaries.

## When to use it

- Summarize a long PDF, report, or article.
- Ask specific questions about a document's contents.
- Compare information across multiple attached documents.
- Extract key points or data from a file.

## Call it from the slash command system

Type `/document` in the composer, select the skill, then attach a file and ask your question.

## Ask the assistant directly

Pro users can also ask the assistant to use the Document skill in plain language without opening the slash menu.
