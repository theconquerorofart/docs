Slash commands are the fastest way to use AscentialxAI's built-in skills. Type `/` in the composer to open the command menu.

## How it works

When you type `/`, a menu appears showing available commands organized by category. Select a command to lock the next response to that skill, or continue typing to narrow the list.

## Command categories

- **Skills** — Built-in tools like memory, vision, scraping, and browser tasks.
- **Prompts** — Saved prompt cards from your Prompt Library.
- **Automations** — Saved automation flows.
- **Recent** — Commands you've used recently.
- **Favorites** — Commands you've starred.

## Commands

Slash commands can include:

| Command | What it does |
| --- | --- |
| `/memory` | Save and recall user facts and preferences. |
| `/vision` | Analyze images, screenshots, or visible screen content. |
| `/scraper` | Extract text, structured data, and metadata from the current page. |
| `/document` | Ask questions about or summarize attached documents. |
| `/browser` | Perform browser tasks like navigating, clicking, typing, and extracting from the current page. |
| `/flow` | Ask the assistant to create a draft automation flow. |
| `/scheduler` | Create, list, and delete scheduled jobs. |
| `/trigger` | List and launch eligible automation flows. |
| `/forge` | Create a custom skill with your own schema and code. |
| `/tune` | Rename or refine a forged skill. |
| `/prune` | List usage and delete forged skills. |

Some commands may require Pro access. See [Plans and access](/plans-and-access) for details.

## Selecting a command

Choose a command from the menu or type the full command (for example, `/browser`) and press **Enter**. The assistant routes your request to the appropriate skill and returns the result in chat.
