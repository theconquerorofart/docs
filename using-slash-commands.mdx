Slash commands give you a quick menu inside the chat composer. Choose what you want to launch, then continue the request in chat.

## Open the command menu

Type `/` in the composer. The menu appears above the composer and filters as you keep typing, and lets you switch categories to narrow what it shows.

## What the menu includes

| Category | What it contains |
| --- | --- |
| Skills | Built-in skill chapters you can use from chat. |
| Prompts | Saved prompt cards from your Prompt Library. |
| Automations | Saved automation flows you can run from chat. |
| Recent | Items you've used recently. |
| Favorites | Items you've starred for quick access. |

## Choose a skill

Select a skill to lock your next message to that skill. After selecting it, write what you want the skill to do and send the message.

Skills have their own pages in the Skills section.

## Apply a prompt

Saved prompts appear in the **Prompts** category. Select a prompt to fill the composer with that prompt's text.

If the prompt includes variables, AscentialxAI asks you to fill them before applying it.

## Run an automation

Saved automation flows appear in the **Automations** category. Select an automation to run it from chat.

Draft flows do not appear here until they are approved and saved.

## Use recent items and favorites

Use **Recent** to reuse commands, prompts, or automations you've used recently.

Use **Favorites** for items you've starred and want to access quickly.
