The Trigger skill lists and launches automation flows configured with an Agent Execution trigger directly from chat.

## When to use it

- Launch a pre-built flow from chat while you're in a conversation, without switching to the Automation Hub tab.

## Call it from the slash command system

Type `/trigger` in the composer, select the skill, then choose which flow to run from the list.

## Ask the assistant directly

Pro users can also ask the assistant to use the Trigger skill in plain language without opening the slash menu.
