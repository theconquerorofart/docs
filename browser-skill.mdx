The Browser skill lets the assistant work directly with the page you're viewing. It can navigate, interact with elements, and extract information from the visible page.

## When to use it

- Navigate to a specific URL.
- Click buttons, fill forms, or type into fields.
- Extract text or structured data from the visible page.
- Scroll, inspect sections, or find specific content.

## Call it from the slash command system

Type `/browser` in the composer, select the skill, then describe what you want to do on the current page.

## Ask the assistant directly

Pro users can also ask the assistant to use the Browser skill in plain language without opening the slash menu.
