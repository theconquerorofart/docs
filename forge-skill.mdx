The Forge skill lets you create custom skills that the assistant can use, going beyond the built-in commands.

## Use Cases

- Creating custom skills when the built-in ones don't cover your exact workflow.
- Packaging a multi-step task into a single reusable command.

## Call it from the slash command system

Type `/forge` in the composer, select the skill, then describe what the new skill should do.

## Ask the assistant directly

Pro users can also ask the assistant to use the Forge skill in plain language without opening the slash menu.
