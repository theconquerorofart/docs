The Prompt Library lets you save prompt templates so you can reuse them anytime.

## Open the Prompt Library

Select the **Prompt Library** tab at the bottom of the side panel.

## Create a prompt

1. Click **New Prompt**.
2. Enter a **title** and optional **description**.
3. Write your prompt **instructions**.
4. Use `{{variable}}` for dynamic values — the assistant will ask you to fill these in when you apply the prompt.
5. Optionally, set a **default model** to use when the prompt is applied.
6. Save.

## Apply a prompt

Select a prompt from the library and choose **Apply**. The prompt fills the chat composer, and any variables are prompted for input. If the prompt has a default model, the model selector switches to it.

## Manage prompts

- **Preview** — See how a prompt looks before applying it.
- **Edit** — Change the title, instructions, variables, or default model.
- **Delete** — Remove a prompt you no longer need.
