The Flow skill creates draft automation flows based on a natural language description. Describe the workflow you want, and the assistant builds it for you.

## When to use it

- Describe a repetitive workflow — like "every time I visit a competitor page, save a screenshot and extract the pricing" — and have the assistant draft the full flow.
- Prototype a multi-step automation without opening the Automation Hub builder, then review and refine the result.

## Call it from the slash command system

Type `/flow` in the composer, select the skill, then describe the workflow you want to automate.

## Ask the assistant directly

Pro users can also ask the assistant to use the Flow skill in plain language without opening the slash menu.
