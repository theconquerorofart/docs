The Scheduler skill lets you schedule automation flows to run at specific times or intervals, backed by Chrome alarms.

## Use Cases

- Running automation flows on a recurring schedule without manual triggers.
- Setting up or checking scheduled jobs from within a conversation.

## Call it from the slash command system

Type `/scheduler` in the composer, select the skill, then describe when you want the flow to run.

## Ask the assistant directly

Pro users can also ask the assistant to use the Scheduler skill in plain language without opening the slash menu.
