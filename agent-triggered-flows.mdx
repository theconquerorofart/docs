Flows configured with an **Agent Execution** trigger can be launched from chat whenever you need them.

## Create an agent-triggered flow

When creating a flow in Automation Hub, choose **Agent Execution** as the trigger. The assistant will then be able to launch it on request.

## Launch a flow from chat

1. Ask the assistant to run an eligible flow.
2. The assistant shows a list of eligible enabled flows.
3. Select the flow you want to run.

The flow executes using the same steps and settings you configured in Automation Hub. Results appear in the chat and are recorded in Execution History.

## Managing triggered flows

All flows — including agent-triggered ones — can be managed from the Automation Hub. You can edit, enable, disable, or delete them just like any other flow.
